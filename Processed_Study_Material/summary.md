## 🔁 Top Repeated Questions Across All Weeks

### Pattern 1: Definition Questions
- *What is Business Development?* → Creating strategic partnerships for long-term value
- *What is Brand Equity?* → Premium value over generic equivalent
- *What is Cultural Blinds?* → Mindsets blocking cross-cultural interaction
- *What is a Stuck Industry?* → Fragmented + fails to evolve

### Pattern 2: Distinguishing Similar Concepts
- Sales vs Marketing vs BD
- Customer vs Consumer
- Outsourcing vs Offshoring
- JV vs Subsidiary
- Basic vs Applied vs Developmental Research

### Pattern 3: 'Which is NOT' Questions
- Which is NOT an endgame strategy? → **Innovation** (correct: Leadership, Niche, Harvesting, Divestment)
- Which is NOT a state of demand? → **Hyper Demand**

### Pattern 4: Stage/Phase Identification
- Competitive Turbulence → **Maturity** phase
- First-mover advantage → **Introduction** phase

### Pattern 5: Case Study Application
- Nirma → Mass-market disruption via price + promotion
- Honda → Autonomous Localization
- Happiest Minds → Digital-native IT startup
- Orchid Chemicals → Niche + technology pharma transformation

## 🎯 Exam Strategy

### Before the Exam
1. **Review Week 1 thoroughly** — foundation and questions appear in every mock exam
2. **Master the definitions** — most traps involve subtle definitional differences
3. **Study case studies** — Nirma, Honda, Happiest Minds, Orchid each teach a strategic principle
4. **Review NOT questions** — memorize which options do NOT belong to each framework

### During the Exam
1. **Read all options** — distractors are crafted to sound correct
2. **Identify question type:** Definition / Distinction / Stage / Case / Framework
3. **Use elimination** — remove obviously wrong options first
4. **Trust frequency patterns** — strategically sound answers are usually correct

### High-Confidence Topics
- Week 1: BD definitions, 8 demand states, strategy roll-out
- Week 2: POC formula, SMART goals, Nirma case
- Week 10: Outsourcing/Offshoring, R&D types, Dual Concerns Model
- Week 11: PLC stages, Competitive Turbulence, endgame strategies
